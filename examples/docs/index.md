# Phoenix DevOps Demo

This example page exists to demonstrate the documentation server and the CI/CD
pipeline in a simple, non-critical setting.

## What this page shows

- Markdown is rendered as styled HTML
- a summary card is generated automatically
- headings are detected and listed
- the first paragraph becomes the quick excerpt

## Why it is useful

The example content gives the project a stable demo target for local testing,
presentations, and pipeline verification.
